# 🔒 Security Policy

### Supported Versions

#### These versions are supproted:

| Version | Supported          |
| ------- | ------------------ |
| 2.7.x   | :white_check_mark: |
| 1.8.x   | :x:                |
| 2.0.x   | :white_check_mark: |
| < 2.0   | :x:                |

### Reporting a Vulnerability

🛑 If you've identified a security issue or vulnerability, report it by contacting us directly in [**Issues**](https://github.com/haykif/Apple_products_web_site/issues) section.

#### Please include the following details:

- A clear description of the issue
- Steps or proof-of-concept to reproduce the vulnerability
- Impact and severity estimation
- Any relevant screenshots or logs (if available)

### Response Time

We aim to respond to security reports within **48 hours**.
